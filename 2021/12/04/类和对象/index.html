类和对象

C++面向对象的三大特性是：封装、继承和多态。

C++认为万事万物皆为对象，对象上有其属性和行为。

例如：

人作为对象，其属性有姓名、年龄、身高、体重...，行为有走、跑、跳、吃饭、唱歌...
车也可以作为一种对象，其属性有轮胎、方向盘、车灯...，行为有载人、放音乐、开空调...

具有相同性质的对象，我们都可以将其抽象称为类，人属于人类，车属于车类。

封装：

封装是C++面向对象三大特性之一

封装的意义：

1、将属性和行为作为一个整体，表现生活中的事物

2、将属性和行为加以权限控制。

在设计类的时候，属性和行为写在一起，表现事物

语法： class 类名 {访问权限： 属性/行为};

例1：

#include<iostream>
#include<string>
#define PI 3.14
using namespace std;

//创建一个圆类 
class circle
{
    //访问权限
    //公共权限 
public:

    //属性
    //半径 
    int m_r;

    //行为
    //获取圆的周长 
    double calculateR()
    {
        return 2*PI*m_r;
    } 
};

在主函数中进行调用

//通过圆类 创建具体的圆（对象） 
circle c1;
//给圆对象的属性进行赋值 
c1.m_r=10;
//获取圆的周长 
cout<<"c1的周长是："<<c1.calculateR()<<endl;

例2：

//创建一个学生类 
class student
{
    //权限 
    //公共权限 
public:

    //属性    
    //姓名 
    string s_name;
    //学号
    string s_ID;

    //行为
    //录入学生信息
    void initStu()
    {
        string name,ID;
        cout<<"请输入学生姓名："<<endl;
        cin>>name;
        s_name=name;
        cout<<"请输入学生学号："<<endl;
        cin>>ID;
        s_ID=ID;
     } 
     //显示学生信息
      void showStu()
      {
        cout<<"姓名："<<s_name<<endl;
        cout<<"学号："<<s_ID<<endl;    
      } 
};

在主函数中进行调用

  //创建一个具体的学生对象    
  student stu;
  //录入学生信息 
  stu.initStu();
  //显示学生信息 
  stu.showStu();

注意：

1、类的定义中，大括号末尾要加分号“；”

2、类的引用操作为“.”
