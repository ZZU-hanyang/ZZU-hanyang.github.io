函数默认参数

函数在声明或者实现的时候可以有默认参数

举例：

int func(int a,int b=10,int c=20)
{
    return a+b+c;
}

在主函数中进行调用
cout<<func(10)<<endl; //输出的结果是10+10+20=40 
cout<<func(10,20)<<endl;//输出的结果是10+20+20=50

如果我们自己传入数据，那么函数体就调用传入的数据，否则就用默认值

注意事项

1、在函数的参数中，如果某个位置的参数有了默认值，那么这个位置之后的所有参数都必须有默认值

例如：
 func(int a,int b=10,int c){...} 是错误的，因为b有了默认参数，那么c也必须有

2、函数的声明和实现只能有一方有默认参数。即函数在声明时有了默认参数，那么在函数实现的时候就不能有默认参数；同样，在函数实现的时候有了默认参数，那么在函数声明的时候就不能有默认参数。

    例如：
//声明一个函数func()
int func(int a=10,int b=20);

//func函数的实现
int func(int a=10,int b=20)
{
    return a+b;
} 
或者
int func(int a=1000,int b=1000)
{
    return a+b;
} 
这样写是错误的
