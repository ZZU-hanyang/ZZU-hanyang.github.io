指针

指针的作用

可以通过指针间接访问内存

注意：

1.内存编号一般是从0开始记录的，一般用16进制数字表示

2.可以利用指针变量保存地址

指针的定义和使用

1.语法：数据类型 * 变量名

例如：

int a = 10 ;
int *p;
p = &a;
   // 或者 int *p = &a;

2.指针变量的赋值

p1=&a;//将a的地址赋给p1
cout<<"a的地址："<<&a<<endl;
cout<<"p1的值："<<p1<<endl;

3.指针变量的解引用

cout<<"指针p1指向的数据："<<*p1<<endl;

4.指针占用的内存空间

cout<<"指针p1占用的内存空间："<<sizeof(p1)<<endl;
cout<<"整型指针占用的内存空间："<<sizeof(int *)<<endl;
cout<<"float型指针占用的内存空间："<<sizeof(float *)<<endl;
cout<<"double型指针占用的内存空间："<<sizeof(double *)<<endl;

指针变量占用的内存空间不随指向的数据类型的变化而变化。

空指针

指针变量指向内存地址编号为0的空间

用途：初始化指针变量

注意：

1.内存地址编号在0~255为系统占用内存，不允许访问

2.访问空指针会报错

int *p = NULL;//定义一个空指针

野指针

指针变量指向了非法的内存空间

例如：

int *p；
p = 0x1100;//指针p指向的内存空间未经过申请，这个指针是野指针

注意：程序设计中应尽量避免出现野指针，用野指针进行访问会报错

const

int a=10;
int b=20;

常量指针

特点：指针的指向可以被修改，但是指针指向的值不能被修改(是指不能通过解引用的方式修改指针指向的内容)

const int *p1=&a;

/*
    a=20;//将20赋给变量a是合法的
    *p1=20//利用解引用的方式改变p1指向的内容是非法的
    p1=&b//可以改变指针p1的指向，让其指向其他内存区域 
*/

指针常量

特点：指针的指向是不可以被修改的，但是可以用指针修改指针指向的内容

int *const p2=&a;
/*
    p2=&b; //将指针指向p2是非法的
    *p2=20;//用解引用的方式改变指针p2指向的内存空间的数据是合法的 
*/

const既修饰指针也修饰变量

const int *const p3 = &a;

指针的指向不能被修改，也不能用解引用的方式修改指针指向的内容

总结：

const修饰哪个，哪个就不能被修改。
