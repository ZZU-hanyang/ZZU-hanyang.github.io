函数的分文件编写

1.创建.h后缀名的头文件

2.创建.cpp后缀名的源文件

3.在头文件中对函数进行声明

4.在源文件中对函数进行定义

示例

max.h

#include

using namespace std;

void max(int num1,int num2);

max.cpp

#include “max.h”

void max(int num1,int num2)
{
    cout<< (num1 > num2 ? num1 : num2)<<endl;
}

main.cpp

#include

#include “max.h”

using namespace std;

int main()
{
    int a=10,b=20;
  max(a,b);
}

注意：加粗部分为重点，与C语言有所不同，这里没用#ifndef和#endif，并且在函数源文件中需要调用该函数对应的头文件，因为该头文件已经调用了#include和using namespace std ,因此在函数的调用中不需要再次进行书写
