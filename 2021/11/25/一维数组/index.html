一维数组

数组的特点：

1.数组中的每个数据都是相同的数据类型

2.数据是由连续的内存位置组成的

3.数组的下标是从0开始索引的

数组的三种定义方式

1.数据类型 数据名 [数组长度] ;

    int array[10];//定义一个长度为10的数组

2.数据类型 数据名 [数组长度] = {值1，值2，值3…};

    int array[10]={10,20,30};

注意：10、20、30会被依次赋给a[0]、a[1]、a[2]，其他未被赋值的会自动被0填充

3.数据类型 数据名 [] = {值1，值2，值3};

    int array[]={10,20,30};

这种赋值方式下，系统会自动计算出数组长度。这个数组的长度为3。
一种计算数组长度的方法是:

cout<<sizeof(array)/sizeof(array[0])<<endl;

数组名的用途

int array[]={10,20,30};

1.可以通过数组名统计数组占用的内存大小

cout<<"整个数组占用的内存空间为："<<sizeof(array)<<endl;
cout<<"每个元素占用的内存空间为："<<sizeof(array[0])<<endl;
cout<<"数组中的元素个数为："<<sizeof(array)/sizeof(array[0])<<endl;

2.可以通过数组名查看数组首地址

cout<<"数组的首地址为："<<array<<endl;
cout<<"数组第一个元素的地址为："<<&array[0]<<endl;

注意：数组名是一个常量，它指向一段内存空间，不可以进行赋值操作

//元素逆置
int arr[]={1,2,3,4,5}; 

int i,j; 
j=sizeof(arr)/sizeof(arr[0])-1;////i和j分别代表要交换的两个数组元素的左右下标
for(i=0;i<j;i++,j--)//循环条件是左下标小于右下标
{
    int tem;
    tem=arr[i];
    arr[i]=arr[j];
    arr[j]=tem;    
}
